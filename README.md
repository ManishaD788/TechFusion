# [Your Club Name] — College Club Website

**A simple multi-page club website built using HTML & CSS**  
This project is a basic static website for a college club, with pages for Home, About, Events, and Join form. It demonstrates semantic HTML, CSS layout using float & Flexbox, and a clean folder structure — ideal for hosting on GitHub Pages or sharing as a portfolio project.

---

## 📄 Overview

This repository contains:

- `index.html` — Home page  
- `about.html` — About Club page (club description + image)  
- `events.html` — Events / Activities page (with event “cards” arranged using CSS Flexbox)  
- `join.html` — Join Club page (a HTML form for new members)  
- `css/style.css` — Common stylesheet for all pages  
- `images/` — Folder for logo, about-page image, and any event images  

The site uses semantic HTML tags (`<header>`, `<nav>`, `<main>`, `<footer>`, etc.), a common navigation bar on all pages, and layout techniques including **float** (for image + text side-by-side on About page) and **Flexbox** (for aligning event cards on Events page).
